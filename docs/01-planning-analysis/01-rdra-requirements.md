### 3.2 ユースケース詳細

#### UC-001: 音声問い合わせを受付する

| 項目 | 内容 |
|------|------|
| アクター | オペレーター |
| 事前条件 | ログイン済み |
| トリガー | 顧客からの電話着信 |
| 基本フロー | 1. 音声入力開始<br>2. リアルタイム認識<br>3. 類似検索<br>4. 回答提示<br>5. 確定・記録 |
| 事後条件 | 対応履歴が保存される |
| ビジネスルール | BR-001, BR-002 |
